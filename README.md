# moodify-app-1
the app for time table

Moodify is a modern web application designed to help users manage their daily and weekly schedules efficiently.

## Features
- User Login
- Event Creation
- Daily & Weekly View
- Event Editing & Deletion
- Clean UI with Tailwind CSS

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS

## Setup

```bash
npm install
npm run dev
